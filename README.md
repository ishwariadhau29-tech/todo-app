# 📝 Todo App
This is a basic Todo App developed using HTML, CSS, and JavaScript. It includes features like adding tasks, marking tasks as completed using checkboxes, deleting completed tasks, preventing duplicate entries, displaying an empty state message, and tracking pending tasks. The project focuses on DOM manipulation and event handling concepts.


A simple Todo App built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add new tasks
- Mark tasks as completed (checkbox)
- Remove completed tasks
- Prevent duplicate tasks
- Show pending task count
- Empty state message when no tasks are present
- Add task using Enter key

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (DOM Manipulation)

## 📂 How to Run
1. Download or clone the repository
2. Open index.html in your browser

## 🎯 Purpose
This project was created to practice JavaScript concepts like DOM manipulation, event handling, and basic UI logic.
